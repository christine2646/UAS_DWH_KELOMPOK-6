Deskripsi Proyek
Proyek ini mengimplementasikan Data Warehouse dengan optimasi ETL Pipeline untuk analisis data kesehatan global, khususnya HIV/AIDS dan kesehatan reproduksi berbasis OLAP. Data yang digunakan berasal dari World Bank Health Nutrition and Population Statistics dengan rentang waktu 2020–2024.

Data kesehatan global memiliki karakteristik multidimensi, time-series, serta banyak missing value, sehingga membutuhkan proses pengolahan data yang terstruktur. Oleh karena itu, dibangun sistem Data Warehouse untuk mengintegrasikan data dari berbagai negara dan indikator kesehatan agar dapat dianalisis secara lebih efisien dan konsisten.

Pendekatan OLAP digunakan untuk mendukung analisis multidimensi terhadap tren HIV/AIDS, perbandingan antar negara, serta perkembangan indikator kesehatan reproduksi secara global.
